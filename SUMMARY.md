# Project Summary: Guru Nanak Sikh Gurdwara Website

This document summarizes the initial development and deployment of the website for **Guru Nanak Sikh Gurdwara**, Murfreesboro, TN.

## Project Overview
- **Objective:** Create a modern, minimalist central hub for the Gurdwara's Sangat.
- **Address:** 343 Rucker Road, Murfreesboro, TN 37127.
- **Tech Stack:** Next.js (App Router), TypeScript, Tailwind CSS.
- **Hosting:** GitHub Pages via GitHub Actions.

## Key Features Implemented (MVP)
1.  **Home Page:** Inviting hero section, weekly Diwan timings (Sundays 10 AM - 1 PM), and location details.
2.  **Events Page:** Schedule for regular Sunday Diwans and monthly Istri Satsang programs.
3.  **Langar & Seva:** Information on community service with a functional layout for Langar sponsorship requests.
4.  **About Us:** Mission statement, core Sikh principles (Naam Japna, Kirat Karni, Vand Chhakna), and visitor guidelines.
5.  **Donations:** Information for in-person and mail-in contributions, with placeholders for future online integration.
6.  **Live Kirtan:** Integration-ready layout for YouTube/Facebook live streams.

## Technical Milestones
- Initialized Next.js project with a customized "Modern & Minimalist" theme.
- Developed reusable `Navbar` and `Footer` components.
- Configured static export (`output: 'export'`) for compatibility with GitHub Pages.
- Created an automated CI/CD pipeline using GitHub Actions (`.github/workflows/nextjs.yml`).
- Successfully deployed to: [https://manmeet3591.github.io/gurdwara-website/](https://manmeet3591.github.io/gurdwara-website/)
